## Prerequisites

- Git
- NodeJs
- pnpm

## Getting Started

- Install dependencies

```sh
pnpm i
```

- Make sure [configs](./src/configs/) is updated
  - Fill in address details in [src/configs/contracts.config.ts](./src/configs/contracts.config.ts)

## Run Dev

```sh
pnpm run dev
```

## Build Prod

```sh
pnpm run build
```

```sh
pnpm run preview
```
